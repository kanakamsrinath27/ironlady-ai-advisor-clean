# Iron Lady AI Career Advisor

## Overview
The Iron Lady AI Career Advisor is an AI-powered customer interaction solution designed to support women learners during the early stage of their career journey. It helps users understand suitable programs, learning paths, and next steps offered by Iron Lady.

## Problem Statement
Many users visiting Iron Lady’s platform are unsure which program best fits their background, experience level, and career goals. This confusion can lead to poor engagement and drop-offs.

## Solution
This project provides an AI-based career advisor that:
- Collects basic user information (background, goals, experience)
- Uses AI to recommend suitable learning paths
- Explains the learning journey in a clear and user-friendly way
- Improves customer engagement and decision-making

## Features
- AI-powered career recommendation
- Simple and intuitive user interface
- Backend API for AI processing
- Graceful fallback response when AI quota is unavailable

## Tech Stack
**Frontend**
- React (Vite)
- Axios
- CSS

**Backend**
- Node.js
- Express.js
- OpenAI API

## How It Works
1. User enters background, career goal, and experience level
2. Frontend sends data to backend API
3. Backend processes input using AI logic
4. Personalized recommendation is displayed to the user
